A personal C# learning repository containing examples and practice programs covering C# fundamentals, Object-Oriented Programming (OOP), Collections, Delegates, Events, Exception Handling, Generics, and more.

## Purpose

This repository is created to:

* Learn C# fundamentals.
* Practice core programming concepts.
* Understand .NET collections.
* Learn delegates and events in depth.
* Build a strong foundation before moving to ASP.NET Core and advanced .NET development.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/hardikmishra24/CSharp_Learning.git
```

2. Open the solution in Visual Studio.

3. Select the desired example from `Program.cs`.

4. Run the project.
